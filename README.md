# MCP Andbox React App

This is a simple React application created using the latest Vite build tool.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Features
- React 18
- Vite for fast development and build
